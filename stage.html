2. Redressement Statistique

Étude et implémentation de modèles mathématiques pour imputer les valeurs manquantes. J'ai déployé une approche hybride : une sélection des variables prédictives via un Arbre de décision, suivie d'une imputation par l'algorithme des K-Plus Proches Voisins (K-NN) basé sur ces variables clés.

Association aux Compétences Nationales :

  • Pôle 3 : Modéliser (EMS)
    AC 42.01 : Mettre en œuvre des modèles de régression — Utilisation de la modélisation pour estimer des tendances sur les variables quantitatives manquantes.
  • Pôle 3 : Modéliser (EMS)
    AC 42.02 : Mettre en œuvre des méthodes d'apprentissage statistique — Configuration et exécution de l'algorithme non-paramétrique des K-NN combiné à un arbre de décision.
  • Pôle 3 : Modéliser (EMS)
    AC 42.03 : Évaluer la qualité et comparer les performances de plusieurs modèles — Analyse comparative de l'impact des biais d'imputation (Hot Deck vs K-NN) sur les distributions marginales.

# --- 1. SÉLECTION DES VARIABLES (ARBRE DE DÉCISION) ---
library(rpart)
library(VIM)

# Entraînement de l'arbre pour identifier les variables explicatives clés
modele_arbre <- rpart(variable_a_imputer ~ ., 
                      data = donnees_brutes, 
                      method = "class", 
                      control = rpart.control(cp = 0.01))

# Récupération automatique des variables importantes de l'arbre
var_predictives <- names(modele_arbre$variable.importance)

# --- 2. IMPUTATION K-NN BASÉE SUR L'ARBRE ---
# Redressement avec les 5 plus proches voisins selon les variables ciblées
donnees_imputees <- kNN(donnees_brutes, 
                        variable = "variable_a_imputer", 
                        k = 5, 
                        dist_var = var_predictives, 
                        imp_var = FALSE)

print("✓ Imputation K-NN post-arbre terminée avec succès.")
